# Security Policy

## Supported Versions

Only the latest commit on `main` is supported with security fixes. This is a
zero-backend, fully client-side static site published to GitHub Pages, so
"versions" are effectively whatever is currently deployed.

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security reports.**

Use GitHub's private vulnerability reporting:
👉 https://github.com/ievsantillan/edm-fabric-usergroup_raffle-spinner/security/advisories/new

You can expect:

- An acknowledgement within **7 days**
- A status update within **30 days**
- Coordinated disclosure once a fix is shipped or the issue is closed as not a
  vulnerability (e.g. out of scope)

## Scope

In scope:

- The deployed site at `https://ievsantillan.github.io/edm-fabric-usergroup_raffle-spinner/`
- Source code in this repository
- The CI/CD workflows under `.github/workflows/`

Out of scope:

- Issues affecting only forks or older commits that have since been patched on `main`
- Social-engineering of maintainers
- Denial of service against the GitHub Pages host (a GitHub infrastructure concern)
- Vulnerabilities in third-party browsers, OSes, or assistive technologies

## Hall of Fame

We don't run a paid bounty, but we're happy to credit reporters in the
released advisory unless you'd prefer to remain anonymous.
